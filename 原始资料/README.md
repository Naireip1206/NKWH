# 原始资料目录

> **不可变来源集合** — LLM 从中读取但永不修改。这是知识库的事实源。

## 使用方式

1. 将新来源（文章、论文、笔记、报告等）放入此目录
2. 告知 LLM 摄取新来源
3. LLM 将读取、总结、并整合到知识库中

## 目录结构

```
原始资料/
├── README.md          # 本文件
├── 资产/              # 图片、附件等媒体文件
└── 来源/              # 已归档的原始文档
```

## 导入方法

- **Obsidian Web Clipper**: 浏览器扩展，一键将网页转为 Markdown 存入此目录
- **手动添加**: 直接复制粘贴或拖入文件
- **批量导入**: 可批量放入多个文件后让 LLM 依次处理
