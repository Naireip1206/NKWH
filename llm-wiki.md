# LLM Wiki — 知识库入口

欢迎！本知识库按照 **LLM Wiki** 方法论搭建。以下是核心速览：

## 三层架构

| 层级 | 位置 | 说明 |
|------|------|------|
| 📦 **原始资料** | `原始资料/` | 不可变的来源文档（LLM 只读不写） |
| 🧠 **知识库** | `知识库/` | LLM 生成和维护的 Wiki 页面 |
| 📋 **模式文档** | [[模式文档.md]] | 定义结构和流程的指令手册 |

## 核心文件

| 文件 | 用途 |
|------|------|
| [[模式文档.md]] | 📖 知识库操作手册 — 告诉 LLM 如何维护 |

| [[原始资料/README.md]] | 📥 如何添加来源 |

## 开始使用

| 你想做什么？ | 方法 |
|-------------|------|
| 📥 **添加新知识** | 放文档到 `原始资料/来源/`，然后告诉我处理 |
| 🔍 **查询已有知识** | 直接提问，我会综合知识库回答 |
| 🧹 **健康检查** | 说"检查知识库"，我会审查并提出改进 |
| 🗂️ **定制结构** | 告诉我你的领域，我们可以调整目录和分类 |
